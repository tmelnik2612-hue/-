📦 ЩО ВСЕРЕДИНІ ПАКЕТУ:

1. minimal_app.py - Мінімальний додаток для Android
2. buildozer_minimal.spec - Конфігурація збірки
3. github_workflow_build-apk.yml - Автоматична збірка на GitHub
4. README_UA.md - Детальна інструкція українською

═══════════════════════════════════════════════════════════════
  🚀 ШВИДКИЙ СТАРТ (5 хвилин)
═══════════════════════════════════════════════════════════════

КРОК 1: Створіть GitHub репозиторій
   → Перейдіть на github.com
   → New repository → Назва: xt-scalperpro
   → Public → Create repository

КРОК 2: Завантажте файли
   → Перейменуйте minimal_app.py → main.py
   → Перейменуйте buildozer_minimal.spec → buildozer.spec
   → Створіть папку .github/workflows/
   → Покладіть туди github_workflow_build-apk.yml
   
   Структура:
   xt-scalperpro/
   ├── main.py
   ├── buildozer.spec
   └── .github/
       └── workflows/
           └── build-apk.yml

КРОК 3: Запустіть збірку
   → GitHub → вкладка "Actions"
   → "Build Android APK" → "Run workflow"
   → Зачекайте 20-30 хвилин ☕

КРОК 4: Скачайте APK
   → Після завершення (зелена галочка ✅)
   → Artifacts → "xt-scalperpro-apk"
   → Скачайте ZIP → Розархівуйте
   → ВАШ APK ГОТОВИЙ! 🎉

═══════════════════════════════════════════════════════════════
  📱 ВСТАНОВЛЕННЯ APK
═══════════════════════════════════════════════════════════════

1. Перенесіть APK на телефон
2. Налаштування → Безпека → "Невідомі джерела" (увімкнути)
3. Натисніть на APK файл
4. Встановити
